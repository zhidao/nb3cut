# nb3cut ver.1.1
                                                by Zhidao
                                                2022. 5. 4. 作成
                                                2022. 5. 6. 最終更新
--------------------------------------------------------------------
### 【説明】

光栄「信長の野望 戦国群雄伝」の顔グラフィックファイルをBMPファイルに
分割するツール。

--------------------------------------------------------------------
### 【コンパイル】

makeすれば実行ファイルnb3cutが出来ます。

--------------------------------------------------------------------
### 【使い方】

カレントディレクトリに
 - palette.nb3
 - Kao.nb3
 - Kao2.nb3
 - Kao3.nb3
を置き、
% nb3cut
とすれば、全顔グラフィックがBMPファイルに出力されます。各ファイル名
は (元ファイル).(3桁の通し番号).bmpとなります。固有顔ファイルには通
し番号の後に名前が付け足されます。
